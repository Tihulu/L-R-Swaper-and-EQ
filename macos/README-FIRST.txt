L/R Swaper Mac

1. Run Install Required Audio Driver.command once if BlackHole is missing.
2. Reboot if BlackHole was installed for the first time.
3. Build or open the app.
4. Choose your real output device.
5. Click Start.

If the Signal meter stays at 0%, enable Microphone permission:
System Settings -> Privacy & Security -> Microphone -> L-R Swaper Mac

Log file:
~/Library/Application Support/LR Swaper Mac/app.log
