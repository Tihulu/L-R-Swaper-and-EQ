# L/R Swaper Linux v4.9

Repository: https://github.com/Tihulu/L-R-Swaper-and-EQ

## Highlights

- Tihuluwave Theme is the default first-launch theme.
- Tihuluwave UI uses PySide6/Qt with a private app venv.
- Plain Theme keeps the original v2.8-style UI.
- Help includes the GitHub repository URL.
- Improved dock launcher identity: `com.tihulu.lr-swaper.desktop`.
- Improved layout polish for Sound Controls, Quick Actions, and L/R Test.
- Visible scrollbars and slider mouse-wheel protection.
- GitHub quick install command:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Tihulu/L-R-Swaper-and-EQ/main/linux/quick-install.sh)
```

## Manual install

```bash
cd linux
chmod +x install.sh check-install.sh
./install.sh
./check-install.sh
lr-swaper
```
