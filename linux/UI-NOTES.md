# L/R Swaper Linux v2.9 UI notes

This package keeps the stable Linux backend and updates the interface layout only.

Feature preservation checklist:

- PipeWire/PulseAudio virtual sink routing is unchanged.
- Target device lock is unchanged.
- Presets remain volume-independent.
- Neutral keeps current volume.
- L/R tests remain available.
- Diagnostics and stream moving tools remain available.
- Closing the app still unloads virtual modules.

The dashboard layout is intended to match the direction of the macOS UI mockup
without reducing Linux functionality.
